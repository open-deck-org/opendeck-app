# OpenDeck — "Aurora" prototype

Design direction: **Aurora** — soft, warm, calming "spa" feel with a slowly drifting ambient
gradient background, generously rounded glass-lite cards, and a warm indigo (`#6C8CFF`) brand tint.

How to view: open `index.html` directly in any modern browser (no build/server needed).
Try Settings → Appearance for live Light/Dark/Auto, tap a card for the Player (with immersive
mode), use the ＋ button to import, and hover/long-press a card to delete.
