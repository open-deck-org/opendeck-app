# OpenDeck — "Liquid Glass" prototype

Design direction: **Liquid Glass** — a faithful CSS approximation of Apple's iOS 26/27 (2026)
look: frosted translucent glass bars and cards floating over blurred colored blobs, specular
hairline edges, large continuous corner radii, and system-blue (#0A84FF) accent.

How to view: open `index.html` directly in any modern browser (double-click it), or serve the
folder and visit it. No build step. Fonts (Inter) load from Google Fonts; icons are inline SVG.
