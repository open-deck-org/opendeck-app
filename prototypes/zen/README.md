# OpenDeck — Zen / Paper

Design direction: **Zen / Paper** — ultra-minimal monochrome (near-black ink on warm off-white paper / near-white on near-black). Maximum whitespace, hairline separators, no shadows, no accent color except a single muted red reserved for destructive actions. Hierarchy is carried entirely by typography (Newsreader serif titles + Inter UI) and space.

To view: open `index.html` directly in any modern browser (no build step). Fonts (Inter, Newsreader) load from Google Fonts and icons from Lucide via CDN, so an internet connection is needed for those assets.
